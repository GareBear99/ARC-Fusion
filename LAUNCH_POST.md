# ARC-Fusion v0.3.0

I built ARC-Fusion: an ARC-native media runtime that uses FFmpeg as a backend while adding binary-first storage, cryptographic manifests, signed receipts, StreamMemory timelines, SURE seeded media recipes, and AI dataset lineage.

It is not a full FFmpeg replacement yet. It is the proof/memory layer that makes media usable inside a verifiable ARC system.
