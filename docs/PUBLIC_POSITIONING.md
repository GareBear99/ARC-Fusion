# Public Positioning

Use this wording:

> ARC-Fusion is a binary-first cryptographic media runtime for ARC systems. It uses FFmpeg as a backend while adding verifiable media memory, signed receipts, StreamMemory timelines, seeded reconstruction recipes, and AI dataset lineage.

Avoid claiming:

- full FFmpeg replacement
- autonomous video understanding
- guaranteed deterministic codec output across all machines
- encrypted storage unless encryption is enabled
- real OCR/transcription until adapters are implemented
