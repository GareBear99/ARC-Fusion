# SURE Media Recipes

SURE recipes represent deterministic media generation states.

Use exact binary storage for canonical source assets and final exports.

Use recipe storage for deterministic procedural artifacts:

- voxel visualizations
- generated preview grids
- waveform visuals
- seeded thumbnails
- deterministic scene maps

A recipe stores:

- generator ID
- seed
- params hash
- params
- optional expected output hash
